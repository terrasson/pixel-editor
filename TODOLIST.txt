[ ] NAVIGATION PAN DANS LA GRILLE ZOOMÉE        🔴 PRIORITÉ HAUTE
    Permettre de se déplacer dans la grille quand on a zoomé,
    sans avoir à utiliser le clic droit (peu intuitif).

    Trackpad Mac (MacBook / Magic Trackpad) :
    - 2 doigts glissés → pan via event "wheel" (deltaX / deltaY)
    - Détecter trackpad via e.deltaMode === 0 + valeurs faibles

    Magic Mouse Apple :
    - Swipe 1 doigt → pan (même API wheel, sensibilité différente)
    - Différencier molette souris (zoom) vs Magic Mouse (pan)

    Mobile / iPad :
    - 2 doigts sans pinch → pan fluide
    - 2 doigts avec écart → zoom (déjà partiel)
    - iPad + Apple Pencil : crayon dessine, 2 doigts panent

    Souris classique :
    - Space + clic glissé → pan (comme Figma / Photoshop)
    - Clic molette (bouton 3) → pan
    - Molette seule → zoom (inchangé)

    Note : cette feature sera naturellement incluse dans la
    migration canvas (voir ci-dessous). Peut être implémentée
    avant, mais sera de toute façon réécrite.
