Plan de collecte d'informations utilisateur et de journalisation
==============================================================

1. Onboarding profil (implémenté ✅)
   - Modal optionnel après connexion + bouton Mon profil (desktop & mobile)
   - Champs : tranche d'âge, genre, pays, région
   - Sauvegarde via window.dbService.saveUserProfile(profile)

2. Écran/section "Mon profil" (implémenté ✅)
   - Accessible via le bouton 👤
   - Modifiable ou ignorable à tout moment
   - Message clair sur l'usage des données

3. Journalisation d'événements clés (implémenté ✅)
   - Helper logUsageEvent(name, payload)
   - Événements : project_saved, profile_saved, profile_prompt_shown, share_option_selected, etc.

4. Dashboard & analyses (à consulter via Supabase Studio)
   - Tables : user_profiles, usage_events
   - Requêtes SQL & export CSV disponibles directement dans Supabase
   - Possibilité future de connecter Metabase / outil BI si besoin

5. Consentement & RGPD
   - Formulaire rappelle que les données sont optionnelles et modifiables
   - TODO : prévoir une action "Supprimer mon compte" si nécessaire
